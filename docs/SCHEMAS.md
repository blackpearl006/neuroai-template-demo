# Data schemas

The fingerprint explorer (`Playground`, `Comparisons`, `ROITable`, `NetworkRadar`)
reads two JSON files from `public/assets/data/`. Regenerate them with
`scripts/build_data.py` (see `scripts/config.py` for inputs). Array lengths must
equal `taxonomy.roiCount` in `content/config.yml`.

## `regions.json`

One object per ROI, in atlas index order (length = `roiCount`).

```json
[
  {
    "id": 1,
    "label": "A8m_L",
    "subregion": "A8m",
    "region": "SFG_L_7_1",
    "our_network7": "Frontoparietal",
    "our_network20": "DMN-2",
    "hemi": "L"
  }
]
```

| Field | Type | Meaning |
|---|---|---|
| `id` | int | 1-based ROI index (matches mesh `roi_<id>` and array positions) |
| `label` | string | short ROI label |
| `subregion` | string | anatomical subregion |
| `region` | string | parent region code |
| `our_network7` | string | 7-network assignment (keys → `NETWORK_COLORS` in `src/lib/themes.js`) |
| `our_network20` | string | finer 20-network/subcortical assignment |
| `hemi` | `"L"`/`"R"` | hemisphere |

## `fingerprints.json` (and per-analysis `fingerprints_<key>.json`)

Nested `analysis → cohort → threshold → { counts, sig }`. `counts` and `sig` are
arrays of length `roiCount`, indexed to `regions.json`.

```json
{
  "main": {
    "ADNI": {
      "top_20_perc_rois": {
        "counts": [0, 0, 12, ..., 100],   // attribution frequency per ROI
        "sig":    [0, 0, 1,  ..., 1]       // 1 = statistically significant
      },
      "top_15_perc_rois": { "counts": [...], "sig": [...] },
      "top_10_perc_rois": { "counts": [...], "sig": [...] },
      "top_5_perc_rois":  { "counts": [...], "sig": [...] }
    },
    "OASIS3": { ... }
  },
  "longitudinal": { ... }
}
```

- **analysis** keys ↔ `taxonomy.analyses[].key`
- **cohort** keys ↔ `taxonomy.cohorts`
- **threshold** keys ↔ `taxonomy.thresholds[].key`

`build_data.py` also writes a split file per analysis (`fingerprints_main.json`, …);
the app lazy-loads these via `loadFingerprintAnalysis(key)` in `src/lib/data.js`.

## Other assets

| Path | What | Generated by |
|---|---|---|
| `public/assets/meshes/atlas.glb` | 3D atlas mesh; named `roi_1`…`roi_N` | (supplied; not script-generated) |
| `public/assets/figures/<COHORT>_<analysis>.png` | glass-brain projections | `make_figures.py` |
| `public/assets/slices/{axial,coronal,sagittal}_<i>.png` | slice renders | `gen_brain_slices.py` |
| `public/assets/preprocessing/<stage>.nii.gz` + `.png` | downsampled pipeline stages | `prep_preprocessing_assets.py` |
