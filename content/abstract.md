### Edit text, not code

Every block of prose on this page lives in a Markdown file in the `content/`
folder. Headings, **bold**, *italic*, [links](https://example.com) and lists all
work — exactly what you already know from writing notes. A typo can never
white-screen the site.

### Numbers live in spreadsheets

The stat cards, the results table and the resource links are plain **CSV** files
you can open in Excel or Google Sheets. Change a value, save, and it shows up.
Brain-atlas data is CSV-driven too.
